 AI Lab (Локальная ИИ-лаборатория)

## О проекте
Этот репозиторий содержит описание и конфигурацию моей домашней среды для запуска нейросетей. Проект создан в рамках самообучения профессии MLOps-инженера.

## ✅ Что уже работает
- **ОС**: Windows 10 + WSL 2 (Ubuntu 24.04)
- **Контейнеризация**: Docker Desktop
- **Движок для LLM**: Ollama (с поддержкой Vulkan для видеокарты AMD Radeon Graphics RYZEN 7 SERIES 4000)
- **Веб-интерфейс**: AnythingLLM (успешно подключен к Ollama)
- **Модели**: 
  - `tinyllama` (тестовая)
  - `llama3.2:3b` (основная, работает в режиме агента)

## 📂 Структура репозитория (в будущем)
- `/docker` - Dockerfile и docker-compose.yml для сервисов
- `/notebooks` - Эксперименты и тесты
- `/docs` - Заметки и инструкции

## 🔧 Полезные команды
- Запуск Ollama: `ollama serve`
- Список контейнеров Docker: `docker ps`
- Вход в WSL: `wsl -d Ubuntu-24.04`
